# 坦克大战 (Tank Battle)

一个用纯 HTML/CSS/JavaScript 实现的经典坦克大战游戏，支持自定义设置、音效系统、主动技能和多种敌人类型。

## 版本说明

- **`坦克大战/tank_battle.html`**: 完整版游戏（推荐）

## 玩法

### 基本操作
- **移动**: 方向键 ↑ ↓ ← →
- **射击**: 空格键
- **暂停**: P 键 或 点击暂停按钮

### 主动技能
- **Q 键 - 护盾**: 激活3秒无敌状态（可强化延长）
- **E 键 - 电磁脉冲**: 减速所有敌人和炮弹50%持续3秒（可强化减速效果）
- **R 键 - 空袭**: 在随机区域投弹造成范围伤害（可强化范围和伤害）
- 进入下一关卡时，所有技能冷却立即刷新

### 游戏目标
- 消灭场上所有敌人坦克即可过关
- 每关消灭普通敌人后会遭遇Boss，Boss关卡同时伴有与当前关卡相同数量和类型的敌人坦克
- 玩家生命值归零则游戏失败
- 进入下一关卡时，技能冷却全部刷新

## 功能特性

### 左侧设置面板

#### 地图选项
- **地形类型**: 平原、沙漠、城市、森林
- **障碍物密度**: 稀疏、适中、密集
- **地图大小**: 超小地图、小地图、中等、大地图、超大地图（影响内容缩放）

#### 玩家设置
- 生命值：1-10（默认2）
- 移动速度：0.5-10（默认2.5）
- 射击冷却：100-3000ms（默认600ms）

#### 敌人设置
- 生命值：1-10（默认3）
- 移动速度：0.5-10（默认1）
- 射击冷却：500-5000ms（默认2000ms）

#### 炮弹设置
- 玩家炮弹速度：1-15（默认4）
- 敌人炮弹速度：1-15（默认3）
- 炮弹伤害：1-10（默认1）

#### 障碍物设置
- 障碍物HP：1-10（默认2）

### 道具系统

#### 基础道具
- ⚡ **移速提升**: 增加移动速度（上限3.0）
- 🔥 **射速提升**: 减少射击冷却（上限200ms）
- 💚 **血量恢复**: 恢复生命值（上限10）
- 💎 **伤害增强**: 增加炮弹伤害（上限+5）
- ⭐ **散弹射击**: 增加散弹数量（上限5）
- 🛡 **临时护盾**: 获得1层护盾，可抵消一次伤害（最多叠加3层）

#### 技能强化道具
- 🛡️ **护盾强化**: 增加护盾持续时间（上限5级）
- 💣 **空袭强化**: 增加空袭范围和伤害（上限5级）
- 🌀 **脉冲强化**: 增加电磁脉冲减速效果（最高90%，上限5级）

### 音效系统
- 发射炮弹音效
- 击败敌人音效
- Boss击败音效
- 道具拾取音效
- Boss来袭警告音效
- 胜利/失败音效
- 技能激活音效
- 护盾抵消音效
- 右上角可切换音效开关

## 游戏元素

### 坦克
- 🔴 **玩家坦克**: 红色，初始HP=2
- 🔵 **普通敌人**: 蓝色，第1关血量1，每2关+1HP
- 🟡 **侦察车**: 黄色，速度快(2x)、体型小(0.7x)、血量低(0.5x)
- 🔴 **重型坦克**: 深红，速度慢(0.5x)、体型大(1.3x)、血量高(3x)
- 🟢 **狙击手**: 绿色，保持距离、精准射击、子弹速度3x
- 🟣 **Boss坦克**: 紫色，血量和伤害随关卡提升

### 敌人解锁机制
- **第1关**: 仅普通坦克
- **第2关起**: 解锁侦察车
- **第4关起**: 解锁重型坦克
- **第6关起**: 解锁狙击手

### 敌人AI行为
- **普通坦克**: 随机移动，偶尔追踪玩家
- **侦察车**: 积极追踪玩家，频繁开火
- **重型坦克**: 稳定追踪，缓慢射击
- **狙击手**: 保持距离，精准射击

### 障碍物
- ⬛ **黑色障碍物**: 不可被摧毁
- 🟩 **绿色障碍物**: 可被摧毁

### 地形效果
- 🟫 **沙地**: 移动减速50%
- 🌲 **森林**: 隐蔽效果，敌人更难发现
- 🌊 **水面**: 坦克无法通行，炮弹可穿过

### 子弹
- 玩家子弹：红色
- 敌人子弹：蓝色
- Boss子弹：较大
- 子弹大小随地图缩放

## Boss系统

- 每关消灭所有普通敌人后出现Boss
- Boss关卡会同时刷新与当前关卡相同数量和类型的敌人坦克
- Boss会清除生成点周围的障碍物
- Boss血量随关卡提升（每关×1.6）
- Boss伤害随关卡提升（每关×1.3）
- Boss具备智能寻路AI

## 运行方式

直接在浏览器中打开 `坦克大战/tank_battle.html` 即可游玩。

## 技术特点

- 纯前端实现，无需服务器
- 使用 Web Audio API 实现音效
- 优化的碰撞检测算法
- 响应式设置面板
- 支持自定义所有游戏参数
- Canvas 渲染引擎
