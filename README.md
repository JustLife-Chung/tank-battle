# 坦克大战 (Tank Battle)

一个用纯 HTML/CSS/JavaScript 实现的经典坦克大战游戏，支持自定义设置和音效系统。

## 版本说明

- **根目录 `index.html`**: 旧版本基础游戏
- **`Version_1/`**: 新版本，包含完整自定义设置、音效系统、优化碰撞检测

## 玩法

### 基本操作
- **移动**: 方向键 ↑ ↓ ← →
- **射击**: 空格键
- **暂停**: P 键 或 点击暂停按钮

### 游戏目标
- 消灭场上所有敌人坦克即可过关
- 玩家生命值归零则游戏失败

## Version_1 新增功能

### 左侧设置面板

#### 玩家设置
- 生命值：1-10（默认2）
- 移动速度：0.5-10（默认2.5）
- 射击冷却：100-3000ms（默认600ms）

#### 敌人设置
- 生命值：1-10（默认3）
- 移动速度：0.5-10（默认1）
- 射击冷却：500-5000ms（默认2000ms）

#### 炮弹设置
- 玩家炮弹速度：1-15（默认4）
- 敌人炮弹速度：1-15（默认3）

#### 障碍物设置
- 绿色障碍物HP：1-10（默认2）

### 游戏控制按钮
- **开始按钮**：点击开始新游戏
- **暂停按钮**：点击暂停/继续游戏

### 音效系统
- 🔊 发射炮弹音效
- 🔊 击败敌人音效
- 🔊 胜利音效
- 🔊 失败音效
- 🔊 被击中音效
- 右上角可切换音效开关

## 游戏元素

### 坦克
- 🔴 **玩家坦克**：红色，初始HP=2
- 🔵 **敌人坦克**：蓝色，初始HP=3，显示血条

### 障碍物
- ⬛ **黑色障碍物**：不可被摧毁，子弹碰上会消失
- 🟩 **绿色障碍物**：可被摧毁（默认2次），子弹碰上会消失

### 子弹
- 玩家子弹：红色
- 敌人子弹：蓝色
- 子弹不能穿过任何障碍物

## 运行方式

1. **Version_1（推荐）**：打开 `Version_1/index.html`
2. **旧版本**：打开根目录 `index.html`

## 技术特点

- 纯前端实现，无需服务器
- 使用 Web Audio API 实现音效
- 优化的碰撞检测算法
- 响应式设置面板
- 支持自定义所有游戏参数
