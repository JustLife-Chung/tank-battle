# HTML/JavaScript 小游戏合集

这是一个用 HTML 和 JavaScript 完成的小游戏合集项目。虽然目前只包含「坦克大战」一个小游戏，但未来会持续更新更多内容。

---

## 坦克大战

用纯 HTML/CSS/JavaScript 实现的经典坦克大战，直接在浏览器中打开即可游玩。

**基本玩法：** 操控红色坦克消灭所有敌人，每关击败普通敌人后挑战 Boss，Boss 关卡同时伴有敌人坦克。

**主要特色：**
- 4 种敌人类型（普通、侦察车、重型坦克、狙击手），随关卡逐步解锁
- 3 个主动技能（护盾、电磁脉冲、空袭），进入下一关自动刷新冷却
- 丰富的道具系统，可强化属性和技能
- 多种地形效果（沙地减速、森林隐蔽、水面阻挡）
- Boss 血量和伤害随关卡指数增长，每关递增难度
- 支持自定义所有游戏参数（地图、属性、难度等）
- Web Audio API 音效系统，无需额外文件

> 详细说明请查看 [坦克大战/README.md](坦克大战/README.md)

## 运行方式

进入对应游戏目录，直接用浏览器打开 HTML 文件即可。
