# 坦克大战

基于 Canvas 的经典坦克大战网页游戏，单 HTML 文件架构，无需任何构建工具或框架。

## 游戏特点

### 基本玩法
- **玩家坦克**（红色）：键盘 ↑↓←→ 移动，空格发射子弹
- **敌人坦克**（蓝色）：自动追踪玩家并射击
- **Boss 战**：每关消灭所有普通敌人后出现 Boss（紫色），击败 Boss 通关
- **关卡递进**：每关敌人数量 +2，Boss 指数级增强

### 障碍物系统
- **不可摧毁障碍物**（黑色）：子弹无法穿透
- **可摧毁障碍物**（绿色）：被击中后扣除 HP，归零后消失，显示受损裂纹

### 道具系统
| 道具 | 效果 | 上限 |
|------|------|------|
| ⚡ 移速提升 | 移动速度 +0.8 | +3.0 |
| 🔥 射速提升 | 射击冷却 -200ms | 最低 200ms |
| 💚 血量恢复 | HP +2 | 不超过上限+5 |
| 💎 伤害增强 | 子弹伤害 +1 | +5 |
| ⭐ 散弹射击 | 额外子弹 +1 | 最多 6 发 |

- 道具拾取后效果**永久保留**，直到被敌人炮弹击中才会清空所有增益
- 道具增益**跨关卡保留**，进入下一关不会消失
- 右侧面板实时显示各项增益等级和上限状态
- 达到上限时拾取会提示"已达到上限"

### Boss 系统
- Boss 体型 1.8 倍、速度 1.5 倍、伤害 2 倍
- Boss HP 公式：`15 × 1.6^(关卡-1)`
- Boss 出场前显示警告提示和音效

### 设置面板（左侧）
- 地形类型：平原、沙漠、城市、森林
- 障碍物密度：稀疏、适中、密集
- 玩家/敌人属性：生命值、移动速度、射击冷却
- 炮弹设置：玩家/敌人炮弹速度、炮弹伤害
- 障碍物 HP 设置
- 游戏控制：开始、暂停按钮
- 音效开关

### 信息面板（右侧）
- 得分、生命、关卡、剩余敌人、Boss HP
- 当前增益等级和上限状态
- 操作说明和游戏规则

## 技术实现

- **单文件架构**：全部代码在一个 HTML 文件中（CSS + JS）
- **Canvas 渲染**：60fps 游戏循环，动态画布尺寸适配屏幕
- **Web Audio API**：程序化生成音效，无需音频文件
- **碰撞检测**：矩形碰撞（AABB）+ 距离²检测（子弹对消）

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑↓←→ | 移动坦克 |
| 空格 | 发射子弹 |
| P | 暂停/继续 |

## 运行方式

直接在浏览器中打开 `tank_battle.html` 即可开始游戏。
